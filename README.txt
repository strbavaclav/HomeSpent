Tento soubor slouží jako REDME k projketu bakalářské práce "Návrh a implementace mobilní aplikace pro správu financí nemovitostí s využítím React Native"
- projekt je detailně popsán v přiložené bakalářské práci



Návod ke spuštěni projektu na lokálním zařízení:

--1.) je nutné aby v zařízení byl nainstalován node.js, npm a expo
--2.) otevřít projekt v příkazové řádce
--3.) příkaz npm install (tímto příkazem se nainstalujou potřebné dependece a moduly, které se na git neverzují.)
--4.) příkaz npm start (tímto příkazem se projekt spustí na lokální zařízení a otevře záložku prohlížeče pro vývojovou konzoli.)


Nejaktuálnější verze projektu je uložena na gitovém uložišti na adrese: https://gitlab.com/strbavaclav/bp_app

PS: autor práce si je vědom některých těžkopádných kódových řešení, které byly způsobeny jeho krátkodobými zkušenosti s frameworkem React Native.
